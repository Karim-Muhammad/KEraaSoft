Simple Clinical CRUD (Doctors, Patients, Admin) with PHP, OOP, MYSQL and MVC Pattern

## Installation

1. Import database `simple_cms_v1.sql`
2. Clone this repository
3. `cd simple-cms`
4. `php -S localhost:<port> -t public/` (`-t` is important)
