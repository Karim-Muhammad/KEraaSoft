Simple PMS CRUD (Categories, Products) and Profile user, with PHP, OOP, MYSQL

## Installation

1. Import database `simple_cms_v1.sql`
2. Clone this repository
3. `cd simple-cms`
4. `php -S localhost:<port> -t public/` (`-t` is important)
